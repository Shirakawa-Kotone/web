# 高考录取数据查询 - Web版

江西省2024-2026年高考录取数据查询工具，支持院校、专业、分数、排名等多维度筛选。

## 项目结构

```text
web/
├── index.html      # 主页面（单页应用）
├── style.css       # 移动端优先的样式
├── app.js          # 核心应用逻辑
├── alldata.js      # 合并后的高考录取数据（~4.7MB）
└── README.md       # 本文件
```

## 功能特性

- **年份切换**：「全部」「2024」「2025」「2026」四种模式
  - 「全部」模式合并同院校+专业组，跨年对比
- **多维度筛选**：省份（多选）、院校名称/代号、选科要求、专业组、分数/排名
- **开关筛选**：仅显示2026招生、隐藏体育专业、隐藏中外合作办学
- **结果排序**：有分数条件时升序，否则降序
- **无限滚动**：每次加载30条，滚动到底部自动加载更多
- **年份差异对比**：自动检测专业组、批次、性质、收费标准的跨年变化
- **备注折叠**：点击展开/收起备注信息
- **首访引导**：第一次使用时自动弹出交互式使用指南
- **回到顶部**：滚动页面后出现浮动按钮

## 数据说明

数据来自江西省2024-2026年高考录取/志愿填报数据，经处理合并为字符串池压缩格式以提高加载效率。

- 总记录数：74,163 条（2024年: 22,472 / 2025年: 24,712 / 2026年: 26,979）
- 数据文件：`alldata.js`（~4.7MB，gzip 后约 1MB）
- 2024-2025年为实际录取数据，2026年为志愿分组参考数据

## 技术栈

纯前端实现，零依赖，可直接托管在 GitHub Pages 上运行：

- HTML5 + CSS3（Flexbox 自适应布局）
- Vanilla JavaScript（无框架、无构建工具）
- 移动端优先设计，适配手机浏览器
