## v0.5.0（2026-06-23）

### 志愿填报辅助系统

- 新增 `REGIONS` 常量（东部沿海、长三角、长江以南/北等 14 个地区）
- 新增选科组合 `AS_SR_LIST` / `AS_SR_CODES`（物化生、物化政、物化地、物政生、物政地、物生地）
- 推荐算法 `calculateTier`：优先 2026 预估排名，回退 2024/2025 平均排名，再回退分数
- 减量匹配 `subjectMatch`（考生选科 ⊇ 专业要求）
- 冲 / 稳 / 保分档逻辑（排名 + 分数双重熔断）
- 地区筛选、选科匹配、专业关键词（多词 OR）搜索
- 仅推荐 2026 年有招生计划的专业组
- 复用 `renderCardGrouped` 渲染推荐结果

### 手机端志愿推荐入口

- 右上角「志愿推荐 / 返回搜索」切换按钮
- 进入时隐藏部分筛选行和年份标签，展开志愿推荐输入区
- 退出时恢复所有筛选行和年份标签
- 桌面端结果栏顶部显示「📋 志愿填报辅助 | 返回搜索」

### 自动退出推荐机制

- 点击「重置」按钮自动退出推荐模式
- 筛选框任意改动自动退出推荐模式

### 返回顶部按钮增强

- 推荐模式下列表滚动超过 300px 显示浮层按钮
- 点击按钮滚动当前活跃列到顶部
- tab 切换时同步检查滚动位置
- 深色模式半透明天依蓝背景

### 2026 招生卡片添加招生简章链接

- 解析 Excel M 列（招生简章链接），存入 `e.sl` 学校链接池
- 展平时通过学校名索引 `schoolLinkPool[r[3]]` 查找链接
- 全部模式 / 单年模式渲染链接行，点击打开阳光高考网
- 新增 `.card-link-row` / `.card-link` 样式

### 冲稳保分档脚本自动计算

- `query.js` 新增 `classify()` 函数按分数 / 排名自动分档
- 传入 `--score` / `--rank` 时 markdown 输出直接分三档
- 冲：bestRank < 考生排名 × 0.92，稳 ≤ 1.08，保：> 1.08
- 纯分数模式：冲 > 考生 + 5，稳 ≥ 考生 - 5，保 < 考生 - 5
- 各档内排序：冲从小到大，稳 / 保从大到小

### query.js 功能增强

- `--format markdown` 输出预排版表格行（院校 / 专业组 / 代号 / 选科 / 批次 / 2024-2025 分排名 / 备注）
- `--keyword` 支持多关键词 OR 筛选
- 默认只输出 2026 年有招生计划的专业组
- 表格新增「包含专业」列，支持调剂参考

### 选科支持扩展

- 支持物化政 / 物化地 / 物生地等组合
- LLM 自动将非物理科目（政 / 地 / 技等）舍弃，提取核心物 / 化 / 生组合
- 新增选科映射表：物化 + 任一第三科 → 物化生，物生 + 任一 → 物生
- 暂不支持历史方向查询

### Skill 收集信息扩展

- 新增批次询问（本科 / 专科 / 提前本科等）
- 新增中外合作办学接受度询问
- 6 个问题分两轮问
- 问题标题加引导语（问句形式）
- 省份多选示例和解析补充（11 条示例）

### docs 文档重构

- `docs/` 纳入版本管理
- `QUERY_GUIDE.md` 全面更新补充
- README 重写，新增安装教程、URL 参数表、AI Skills 指南

## v0.1.0（2026-06-21）

### 微信小程序转 Web 版

将江西省 2024-2026 年高考录取数据查询小程序转换为单页 Web 应用，支持 GitHub Pages 部署，移动端优先。

#### 功能特性

- **单页应用架构**：加载屏 → 引导页 → 主查询界面
- **完整移植筛选功能**
  - 年份切换（全部 / 2024 / 2025 / 2026）
  - 省份多选弹窗
  - 院校代号 / 名称模糊搜索
  - 选科要求、批次、专业组
  - 最低分 / 最低排名范围筛选
- **跨年合并对比模式**，自动检测差异
- **首访交互式引导教程**（跳过标记 `localStorage.tutorial_done`）

#### 数据

- 字符串池压缩格式（`alldata.js`，~4.7MB）
- 总计 74,163 条记录
  - 2024 年：22,472 条
  - 2025 年：24,712 条
  - 2026 年：26,979 条
- 分块异步展开（5000 条 / 块），实时进度条

#### 技术栈

- HTML5 + CSS3（移动端优先，Flexbox 自适应）
- Vanilla JavaScript（无框架、无构建工具）
- GitHub Pages 部署

## 关于版本号

本项目的版本号采用语义化版本（[SemVer](https://semver.org/)）：

- 主版本：架构或数据格式重大变更
- 次版本：功能新增
- 修订号：Bug 修复、样式微调、文档更新

首次发布为 v0.1.0（Web 初始版），后续按此规则递增。
